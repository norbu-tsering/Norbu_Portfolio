About Me

Humble Begining

I love spending free time with my computer since childhood. I self-taught Adobe photoshop and indesign when I was 14 and published a book when I was 15 in Tibetan language. I started making basic websites using HTML and CSS at that time.

I reconnected with web design & development in 2019 while working as a full-time video editor. Not long after, I enrolled in TAFE NSW Hornsby to learn web development while building projects on the side.

Here are few technologies I've been working and learning with recently:

  • CSS/Sass
  • WordPress
  • JavaScript (ES6+)
  • PHP
  • React
  • MySQL
